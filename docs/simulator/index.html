Shepherding simulator

The reactive shepherding library of El-Fiqi et al., IEEE Access 2020 (DOI 10.1109/ACCESS.2020.3037325), compiled unchanged to WebAssembly - running live in your browser
speed ready

Space runs and pauses. Every run is seeded and exactly repeatable; change the seed for a different episode. The dashed circle is the goal region: the task completes when every sheep is inside it.

Model (the IEEE Access comparison)

Regulated forces, distance-based neighbourhood, circular path planning: the model proposed in the paper.

Scenario

parameters (Table 2 of the paper)

Defaults are Table 2 of the paper. The sheepdog sensing range R_beta_pi defaults to 65 (unlimited in practice), matching the paper-era model; the limited-sensing extension added to the library later can be explored by lowering it.
